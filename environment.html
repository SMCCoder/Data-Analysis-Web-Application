Environment

Analyze the world's biggest current environment issues like global warming factors, ocean acidification, coral reef destruction, and invasive species.

Info

Begin by selecting the API of your choice which will retrieve data from the website indicated. Then select a visual and if desired, select a data analysis method. Upon selecting the Red List API, enter a scientific name of an animal in the dropdown field. Options will load as you type. Sorry, the Knoema API is still currently on its way. Then, click 'Get Data' and your selected data will display in the table. To view the visual you selected, select what you want to be on the x- and y-axis. Click 'Graph'.
